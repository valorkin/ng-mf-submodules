## Steps to run this

## **NOTE**
install `yarn`  

### quick start
```bash
npm i
npm start
```

### start in production mode
```bash
npm run start:prod
```

### install
```bash
npm i
```

### build
```bash
npm run build
```

### serve
```bash
npm run serve
```

### start will do clean build and start
```
npm start
```
